# 미로 게임 설명


### 게임 진행 순서
```
1. 게임 시작 버튼을 누른다.
2. 플레이어가 어두운 상황에서 미로의 출구를 찾는다.
3. 탈출에 성공하면 알람과 함께 타이머가 끝난다.
```

### 추후 수정 예정
```
1. 시야 범위 설정
2. 단계별 난이도 설
3. 색 수정
4. 하드코딩하였기 때문에 코드 수정
5. 효과음 추가
6. Eller's algorithm 활용하여 미로 생성
```
